## Project Scaffold

The project scaffold includes:
- Basic NestJS application structure
- Prisma setup
- Environment configuration
- Example module structure

Use this as a starting point for your projects.

## Usage

1. Copy the `project-scaffold/` folder
2. Rename it to your project name
3. Run `npm install`
4. Configure your database connection in `.env`
5. Start building!

## Customization

Customize the scaffold to fit your needs:
- Add additional modules
- Configure authentication
- Set up Swagger
- Add your own features
