# Week 02 — HTTP, REST APIs & Database Introduction (Index)

This week focuses on understanding RESTful API design principles and introducing database concepts.

## Overview

- Focus: REST API principles, HTTP status codes, database fundamentals, and PostgreSQL introduction.
- Outcomes:
  - Understand RESTful API design principles
  - Learn HTTP status codes and when to use them
  - Grasp database fundamentals (what, why, types)
  - Set up PostgreSQL locally or in the cloud
  - Understand the relationship between APIs and databases
