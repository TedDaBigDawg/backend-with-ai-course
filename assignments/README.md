# Assignments

This folder hosts homework, mini projects, and the final capstone project briefs and rubrics.

## Structure

- `mini-project-1.md` — Simple REST API (CRUD operations)
- `mini-project-2.md` — Authentication-enabled API
- `mini-project-3.md` — API with Documentation
- `final-project.md` — Final Capstone Project Brief + Rubric

## Submission

- Submit a GitHub repository link per assignment
- Include a README with instructions and API documentation
- Follow the rubric acceptance criteria
- Use AI tools to learn and understand, but submit your own work

## Grading

- Mini Projects: 40% total (10-15% each)
- Final Project: 25%
- See syllabus for complete breakdown
