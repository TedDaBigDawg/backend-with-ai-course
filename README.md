# Backend Fundamentals with AI (14 Weeks)

A comprehensive beginner-friendly backend development course delivered over 14 weeks (1 class/week, 2 hours each). Students learn core backend concepts using NestJS, Prisma, and PostgreSQL through hands-on lessons, structured exercises, and projects, culminating in a final portfolio-ready backend API. AI assistance is integrated throughout to support learning and problem-solving.

- Duration: 14 weeks (14 sessions, ~28 hours)
- Modality: In-person or remote
- Prerequisites: Basic computer literacy; no coding experience required

## Learning Outcomes

By the end of the course, students will be able to:
- Build RESTful APIs using NestJS framework
- Design and manage databases with PostgreSQL and Prisma ORM
- Implement authentication and authorization with JWT
- Create well-documented APIs with Swagger/OpenAPI
- Handle errors, validation, and security best practices
- Deploy backend applications to production
- Use AI tools effectively as a coding assistant and learning partner
- Ship a complete, production-ready backend API

## Structure

- 14 detailed weekly lesson plans with 120-minute sessions
- 3-4 mini-projects + 1 final capstone project
- Rubrics, checklists, and submission templates
- Student setup instructions and curated resources
- NestJS project scaffold template
- Swagger API documentation examples

## Tech Stack

- **Framework:** NestJS
- **Database:** PostgreSQL
- **ORM:** Prisma
- **Documentation:** Swagger/OpenAPI
- **Authentication:** JWT (JSON Web Tokens)
- **AI Tools:** ChatGPT, Claude, GitHub Copilot (as learning assistants)
